update执行过程
  update a = 3 where id = 1
获得ID=1的这一行 -> 数据是否在内存中? -> 是 -> 返回数据行    -> 将这行的a赋值为3 -> 写入新行 -> 新行更新如入内存 -> 写入redolog处于prepare准备阶段 -> 写 binlog->提交事务,处于commit状态
                  |                             ^
                  |   否                         |
                  |------------->磁盘中获取读入内存-|
